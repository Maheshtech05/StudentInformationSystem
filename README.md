# StudentInformationSystem
Student Information System is a Java-based, menu-driven console application for managing student records. It supports adding, viewing, searching, updating, and deleting students with proper input validation and formatted output, using ArrayList for data management.

## Features
- CRUD operations on student data
- Search by Student ID or Name
- Input validation (age, grade, contact)
- Tabular formatted output
- Sample data for testing

## Tech Stack
- Java (JDK 11+)
- IntelliJ IDEA
- ArrayList

## Project Structure
- src/        → Java source code
- docs/       → Documentation
- test_data/  → Sample data and test cases

## How to Run
### IntelliJ IDEA
1. Open project
2. Mark `src` as Sources Root
3. Run `StudentInformationSystem.java`

### Command Line
```bash
javac src/*.java
java -cp src StudentInformationSystem
